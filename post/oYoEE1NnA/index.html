Windows系统下使用Vmware虚拟Mac OS

2020-01-15

第一步 准备软件

在 Windows系统下安装虚拟 Mac OS 系统之前，你需要先准备好下面这三个软件：

  • VMware Workstation（提供虚拟环境）
  • Unlock master（使VMware可以运行Mac OS）
  • Mac OS镜像系统
    这三个东西，我会放在文章评论里面，有需要的可以自取，VMware没有激活码，自己谷歌一个即可。

第二步 解锁VMware下添加Mac OS的权限

VMware 默认不支持安装 Mac OS 系统，我们需要借助Unlock-master，获得安装 Mac OS系统的权限。下载并解压 Unlock-master软件后，需要将 Unlock-master文件夹放在与 VMware文件夹所在的同一文件夹下，如下图。

打开 Unlock-master 文件夹，找到一个名为win-install.cmd的文件，右键选择以管理员身份运行。等待黑窗口自动运行，等待窗口显示Starting VMware services…，即意味着成功解锁安装 Mac OS 系统的权限。

第三步 添加 Mac OS 虚拟机

首先，运行VMware软件，选择创建新的虚拟机，如下图。

然后，在弹出的新建虚拟机向导窗口，选择典型，继续下一步。

选择安装程序光盘映像文件(iso)，点击右侧的浏览，在弹出的选择文件框中，选择全部文件。

打开完成解压缩的镜像系统 cdr 文件。打开后，会出现无法检测此光盘映像中的操作系统的错误提示，可以略过不管，继续下一步的操作。

客户机操作系统选择Apple Mac OS X，下面选择对应的系统版本，由于我下载的镜像系统版本为 10.12 ，因此版本选择macOS 10.12。

在命名虚拟机的环节，需要设置虚拟机的名称和存放的位置，名称默认即可，位置选择一个空间充足的磁盘。

第四步 启动Mac OS

完成以上操作后，窗口会新增一个 Mac OS 10.12 的标签页，点击工具栏的运行按钮，等待镜像系统完成加载。

镜像系统完成加载后，需要先进行一些基础设置，如设置系统的语言、阅读系统使用协议等，这些略过。
在安装macOS步骤时，点击顶部的实用工具，选择磁盘工具。

在弹出的磁盘工具面板，选择左侧 VMware 开头的磁盘，接着点击顶部的抹掉按钮。

在弹出的面板中，修改默认的名称，下面的格式和方案选项保持默认值即可。

抹除数据之后，关闭磁盘工具面板，可以发现右侧新增了一个 macOS 的磁盘图标，选中磁盘，点击底部的继续。

完成以上所有的操作后，Mac OS系统才正式开始安装，与安装 Windows 系统一样，这是一个漫长的等待过程。

系统完成安装后，还需要进行一定的初始化设置，当提示登录 Apple ID 时，如果你之前没注册过 Apple ID，这里可以选择不登录。

完成一系列初始化设置之后，等待系统进入 Mac OS 桌面，就意味着系统安装成功了。
